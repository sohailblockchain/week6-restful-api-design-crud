
# Simple REST API Demo (Beginner)

## How to Run

1. Install Node.js
2. Make sure MongoDB is running
3. Run:

npm install
npm run dev

Server:
http://localhost:5000

## Endpoints

POST    /products
GET     /products
GET     /products/:id
PUT     /products/:id
DELETE  /products/:id

Everything is inside server.js for easy understanding.
